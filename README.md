# AddressBook
# Developed a menu-driven Address Book application using the C programming language.

# Features:

Add, view, search, edit, and delete contacts

File handling for persistent data storage

Input validation and proper error handling

Modular and structured code design

# Technologies Used:

C Programming, File Handling, Structures, and Pointers

# Additional Highlights:

Used structured data types to efficiently organize contact information

Ensured data integrity through validation checks

Designed a user-friendly console interface

Applied modular programming concepts for better code readability and maintainability
